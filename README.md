# Uber Power BI Dashboard

## Project Overview

This project presents an interactive Power BI dashboard built to analyze Uber ride data and provide actionable business insights.

## Features

* Booking Status Analysis
* Revenue Tracking
* Vehicle Type Performance
* Distance Metrics
* Pickup and Drop Location Analysis
* Customer Rating Analysis
* Driver Rating Analysis
* Interactive Filters and Slicers

## KPIs

* Total Bookings
* Revenue
* Lost Bookings
* Total Distance
* Average Distance
* Customer Ratings
* Driver Ratings

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## Dashboard Insights

* Revenue trends across different periods
* Vehicle-wise performance comparison
* Customer satisfaction metrics
* Operational efficiency indicators
* Booking completion and cancellation analysis

## Skills Demonstrated

* Data Cleaning
* Data Transformation
* Data Modeling
* DAX Calculations
* Dashboard Design
* Business Intelligence Reporting
